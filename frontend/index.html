Requirements · mock vs Docker
  • Single study per upload · default cap 320 MB (SUPREM_MAX_UPLOAD_MB).
  • Real inference: NVIDIA GPU and Docker Desktop (Linux engine); image qchen99/suprem:v1. See README for one-shot scripts/suprem-docker-once.ps1.
  • Screen-demo mode: set SUPREM_INFERENCE_MODE=mock. UI banner shows MOCK (tiny fast outputs).
